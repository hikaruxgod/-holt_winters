# Лабораторная работа 6: Модель Хольта-Уинтерса

## Описание

Реализация мультипликативной модели Хольта-Уинтерса на Python для прогнозирования временных рядов с трендом и сезонностью.

## Возможности

- ✅ Полная реализация модели с нуля (без statsmodels)
- ✅ Автоматический подбор гиперпараметров (α, β, γ, L)
- ✅ Прогнозирование на несколько периодов вперёд
- ✅ Вычисление метрик качества (MAE, RMSE, MAPE)
- ✅ Визуализация результатов

## Использование

### Базовый пример
```python
import pandas as pd
from holt_winters import HoltWintersModel

# Данные
data = pd.Series([362, 385, 432, 341, 382, 409, 498, 387, ...])

# Создание модели
model = HoltWintersModel(alpha=0.3, beta=0.1, gamma=0.4, season_length=4)
model.fit(data)

# Прогноз
forecast = model.predict(steps=4)
print(forecast)
```

### Автоматический подбор параметров
```python
from holt_winters import tune_hyperparameters

best_params, rmse = tune_hyperparameters(data, n_trials=100)
model = HoltWintersModel(**best_params)
model.fit(data)
```

## Математическая модель

**Уровень:**
```
L[t] = α × (Y[t] / S[t-L]) + (1 - α) × (L[t-1] + T[t-1])
```

**Тренд:**
```
T[t] = β × (L[t] - L[t-1]) + (1 - β) × T[t-1]
```

**Сезонность:**
```
S[t] = γ × (Y[t] / L[t]) + (1 - γ) × S[t-L]
```

**Прогноз:**
```
F[t+h] = (L[t] + h × T[t]) × S[t+h-L]
```

## Параметры

- **α (alpha)** - сглаживание уровня (0 < α < 1)
- **β (beta)** - сглаживание тренда (0 < β < 1)
- **γ (gamma)** - сглаживание сезонности (0 < γ < 1)
- **L** - длина сезона (4 для кварталов, 12 для месяцев)

## Результаты

Программа выдаёт:
- Коэффициенты модели (a - уровень, b - тренд)
- Сезонные коэффициенты (F[1], F[2], ..., F[L])
- Прогноз на заданное количество периодов
- Метрики качества (MAE, RMSE, MAPE)
- Графики результатов
